# Hilda Posada — Data Scientist | ML Engineer | Chemistry AI

**MS Organic Chemistry, CSULB 2023. Omdena ML Project Lead. AI Trainer at Outlier.**

I build machine learning systems that solve real problems: churn prediction that identified $1.4M in at-risk revenue, carbon capture models bridging chemistry and climate tech, and drug discovery tools applying deep learning to molecular science.

---

## What I Build

**Predictive ML Systems**
Built an 83% ROC-AUC churn model on 7,043 telecom customers. Identified 1,869 at-risk accounts. Delivered $146K projected annual savings.

**Chemistry + AI**
Predicting CO2 adsorption in Metal-Organic Frameworks (2024 Nobel Prize field) using neural networks trained on 12,020 MOF structures. R² = 0.87.

**NLP and RAG Pipelines**
PDF Knowledge Bot using LlamaIndex, HuggingFace embeddings, and Mistral-7B. Retrieval-augmented generation with source citations.

**Computer Vision**
Animal image classifier trained on 47,000+ images. ResNet-50 transfer learning. 10 exotic species.

**Agentic AI**
Multi-agent research system: Search Agent, Summarizer, Validator, Orchestrator. Built for the Coral Protocol hackathon.
